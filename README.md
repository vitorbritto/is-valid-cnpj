# is Valid CNPJ Number

Validation for CNPJ Number input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var cnpjInput = getElementById(#contact-cnpj);
isValid.cnpj(cnpjInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
